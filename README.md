# Bank_Marketing-UCI-Project
Projeto de machine learning em Python para prever a subscrição de depósitos a prazo com base em dados de marketing bancário. Inclui Análise Exploratória de Dados (EDA), pré-processamento, treino e avaliação de modelos (Logistic Regression, Random Forest, XGBoost) com métricas de desempenho e análise de variáveis mais influentes.
